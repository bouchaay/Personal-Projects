PIM Project: Router with a cache using the Ada programming language.

I recently completed a project in which I designed and implemented a router with a cache using the Ada programming language. The router is responsible for forwarding data packets to their intended destinations, and the cache is used to store frequently accessed data in order to improve the overall performance of the system. The project required a thorough understanding of network protocols and data structures, as well as the use of object-oriented programming concepts in Ada. I faced several challenges throughout the project, such as implementing the cache eviction policy and debugging various issues with the routing algorithms. However, I was able to overcome these challenges through careful planning and testing, and the final product was a functional router with a cache that demonstrated improved performance compared to a traditional router without a cache. I am proud of the skills and knowledge I gained from this project, and I believe it will be valuable experience for future endeavors in the field of computer networking.

Signal Treatment: A modem according to the V21 recommendation of the ITU using Matlab.

I recently completed a project on MATLAB in which I implemented a modem according to the V21 recommendation of the ITU (International Telecommunication Union). The modem is designed to transmit and receive binary data streams over a frequency-modulated signal. One of the main challenges of the project was constructing the frequency-shift keying (FSK) modulated signal. This involved generating a digital signal and scaling the temporal and frequency axes. I also implemented filters to demodulate the signal, using both low-pass and high-pass filters as well as an energy detector. In addition to demodulating the FSK signal, I also implemented a FSK demodulator that is able to handle synchronization errors. This involved using a cosine function to compare the received signal with the expected FSK frequencies, and adjusting the timing accordingly. Overall, this project required a thorough understanding of digital signal processing and filter design, as well as the use of MATLAB's built-in functions and tools. I am proud of the skills and knowledge I gained through this project, and I believe it will be valuable experience for future endeavors in the field of telecommunications.
